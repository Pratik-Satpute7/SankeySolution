
# Task Manager Web App

A simple task manager built using **HTML, CSS, and JavaScript**.  
It allows users to add, edit, complete, and delete tasks.  
Tasks are saved using **LocalStorage**, so they remain after page refresh.

---

## Features
- Add tasks
- Edit tasks
- Mark tasks as completed
- Delete tasks
- Task count and progress tracking
- Random Chuck Norris joke using public API

---

## Technologies Used
- HTML
- CSS
- JavaScript
- Chuck Norris API

---

## How to Run
1. Download or clone the repository  
2. Open `index.html` in a browser
